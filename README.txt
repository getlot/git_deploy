The git_deploy module requires the glip library. The version this is tested 
against is at git://github.com/halstead/glip.git and all updates to glip are
tracked there. glip is licensed under GPLv2.

To install:
1. Clone from git://github.com/halstead/glip.git
   Or download from https://download.github.com/halstead-glip-6f57cd5.tar.gz

2. Place the glip directory at sites/all/libraries/glip so that you have
   sites/all/libraries/glip/lib
